Implementation Guide

How to integrate this design system into your documentation:

Step 1: Include CSS

Add the design system CSS to your documentation:

<link rel="stylesheet" href="design-system.css">
Step 2: Use Components

Apply CSS classes to your HTML elements:

<button class="btn btn-primary">Action</button>
Step 3: Customize

Override CSS custom properties for brand customization:

:root { --luigi-purple-primary: #your-color; }
