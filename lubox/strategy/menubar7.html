Autocomplete Overview

Welcome to Luigi's Box Autocomplete. Our powerful engine provides instant, relevant suggestions to guide your users to the right products, categories, or articles as they type.

Choose Your Integration Path

You can integrate Autocomplete in two ways, depending on your needs.

Recommended

Autocomplete.js Library

The fastest way to get started. Our lightweight JavaScript library handles the UI and API calls for you with minimal setup.
Advanced

Direct API Integration

For full control over the user experience. Call our API from your backend or frontend framework to build a completely custom UI.
